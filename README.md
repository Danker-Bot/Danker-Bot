💻 Somos un **equipo de desarrollo** enfocado en construir soluciones digitales modernas:  
**aplicaciones, sitios web y sistemas personalizados**, con un enfoque en **inteligencia artificial y automatización**.

- 📱 **Aplicaciones móviles y web** multiplataforma  
- 🧠 **Integraciones con IA** (ChatGPT, Gemini, APIs personalizadas)  
- ⚙️ **Sistemas de gestión y automatización**  
- 🌐 **Desarrollo de sitios profesionales y landing pages**
